## 🚀 快速部署

### 方法一: 一键安装 (推荐)

1. **上传文件到青龙面板**
   ```bash
   # 进入青龙面板脚本目录
   cd /ql/scripts
   
   # 上传轻量级版本文件:
   # - qinglong_kanong_lightweight.py
   # - install_lightweight_monitor.sh
   # - test_lightweight_config.py
   ```

2. **执行一键安装**
   ```bash
   chmod +x install_lightweight_monitor.sh
   ./install_lightweight_monitor.sh
   ```

3. **添加青龙面板定时任务**
   - 名称: `卡农论坛轻量级监控`
   - 命令: `python3 /ql/scripts/qinglong_kanong_lightweight.py`
   - 定时规则: `0 */2 * * *` (每2小时执行)

### 方法二: 手动安装

1. **安装依赖**
   ```bash
   # 进入青龙面板容器
   docker exec -it qinglong bash
   
   # 安装基础依赖
   pip3 install requests beautifulsoup4 lxml
   ```

2. **部署脚本**
   ```bash
   # 上传 qinglong_kanong_lightweight.py 到 /ql/scripts/
   chmod +x /ql/scripts/qinglong_kanong_lightweight.py
   ```

3. **测试配置**
   ```bash
   python3 /ql/scripts/test_lightweight_config.py
   ```

## ⚙️ 技术实现原理

### HTTP 协议直接获取

```python
# 创建 HTTP 会话
session = requests.Session()
session.headers.update({
    'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36'
})

# 直接获取页面 HTML
response = session.get(board_url, timeout=30)
html = response.text
```

### HTML 解析策略

#### 方案一: BeautifulSoup 解析 (推荐)
```python
from bs4 import BeautifulSoup

soup = BeautifulSoup(html, 'html.parser')
post_rows = soup.select('tbody tr')

for row in post_rows:
    title_link = row.select_one('a[href*="xyk-"]')
    author_link = row.select_one('a[href*="s-uid-"]')
```

#### 方案二: 正则表达式解析 (备用)
```python
import re

title_pattern = r'<a[^>]*href="([^"]*(?:thread|xyk-)[^"]*)"[^>]*>([^<]+)</a>'
matches = re.findall(title_pattern, html)
```

### 反爬虫对策

1. **真实 User-Agent**: 模拟真实浏览器
2. **Session 保持**: 维持会话状态
3. **请求节流**: 合理控制请求频率
4. **错误重试**: 网络异常自动重试
