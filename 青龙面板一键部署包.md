## 🚀 一键部署步骤

### 步骤 1: 上传文件到青龙面板

1. **进入青龙面板容器**
   ```bash
   docker exec -it qinglong bash
   cd /ql/scripts
   ```

2. **创建项目目录**
   ```bash
   mkdir kanong_monitor
   cd kanong_monitor
   ```

3. **上传所有文件到此目录**
   - 可以通过青龙面板的文件管理功能上传
   - 或者使用 `wget`、`curl` 等命令下载

### 步骤 2: 执行一键安装

```bash
# 给安装脚本执行权限
chmod +x install_qinglong_monitor.sh

# 执行安装
./install_qinglong_monitor.sh
```

### 步骤 3: 测试配置

```bash
# 运行配置测试
python3 test_qinglong_config.py
```

### 步骤 4: 添加定时任务

在青龙面板 Web 界面添加定时任务：

- **任务名称**: `卡农论坛监控`
- **定时规则**: `0 */2 * * *` (每2小时执行)
- **执行命令**: `python3 /ql/scripts/kanong_monitor/qinglong_kanong_monitor.py`

#### 1. Playwright 安装失败
