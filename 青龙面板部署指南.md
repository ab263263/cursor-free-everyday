# 🐉 青龙面板 - 卡农论坛监控部署指南

## 📋 项目概述

本项目为青龙面板定制的卡农论坛自动化监控脚本，集成了浏览器自动化、AI智能分析和邮件通知功能。

### ✨ 核心特性
- 🕷️ **浏览器自动化**: 使用 Playwright 真实浏览器环境抓取
- 🤖 **AI 智能分析**: 集成 Gemini AI 分析帖子价值
- 📧 **整合邮件通知**: 所有分析结果整合为一封邮件发送
- 💾 **数据持久化**: 避免重复处理相同帖子
- 🐉 **青龙面板适配**: 完全适配青龙面板运行环境

## 🚀 快速部署

### 方法一: 自动安装 (推荐)

1. **上传文件到青龙面板**
   ```bash
   # 进入青龙面板脚本目录
   cd /ql/scripts
   
   # 上传以下文件:
   # - qinglong_kanong_monitor.py
   # - install_qinglong_monitor.sh
   ```

2. **执行自动安装脚本**
   ```bash
   chmod +x install_qinglong_monitor.sh
   ./install_qinglong_monitor.sh
   ```

3. **在青龙面板添加定时任务**
   - 名称: `卡农论坛监控`
   - 命令: `python3 /ql/scripts/qinglong_kanong_monitor.py`
   - 定时规则: `0 */2 * * *` (每2小时执行)

### 方法二: 手动安装

1. **安装依赖**
   ```bash
   # 进入青龙面板容器
   docker exec -it qinglong bash
   
   # 安装 Python 依赖
   pip3 install playwright requests asyncio
   
   # 安装浏览器
   playwright install chromium
   ```

2. **上传脚本**
   ```bash
   # 将 qinglong_kanong_monitor.py 上传到 /ql/scripts/
   # 设置执行权限
   chmod +x /ql/scripts/qinglong_kanong_monitor.py
   ```

3. **配置定时任务**
   在青龙面板 Web 界面添加定时任务

## 📅 定时任务配置

### 推荐的定时规则

| 频率 | Cron 表达式 | 说明 |
|------|-------------|------|
| 每2小时 | `0 */2 * * *` | 推荐，平衡及时性和资源消耗 |
| 每4小时 | `0 */4 * * *` | 适合服务器资源有限的情况 |
| 每小时 | `0 * * * *` | 最及时，但消耗较多资源 |
| 工作时间 | `0 9-18/2 * * *` | 仅在工作时间执行 |

### 青龙面板任务配置示例

```bash
# 任务名称
卡农论坛监控

# 定时规则 (每2小时执行)
0 */2 * * *

python3 /ql/scripts/qinglong_kanong_monitor.py

# 任务备注
自动监控卡农论坛最新帖子，AI分析后发送邮件通知
```
